# 🎬 Movie Dashboard

A simple Movie Dashboard built using Python and HTML.

## Features
- Reads movie data from an Excel file
- Displays movie information on a web page
- Beginner-friendly project using Python

## Technologies Used
- Python
- Flask
- HTML
- Pandas
- OpenPyXL

## Project Structure

```
Movie-Dashboard/
│── app.py
│── index.html
│── movies.xlsx
```

## How to Run

1. Install the required libraries

```bash
pip install flask pandas openpyxl
```

2. Run the application

```bash
python app.py
```

3. Open your browser and visit

```
http://127.0.0.1:5000
```
